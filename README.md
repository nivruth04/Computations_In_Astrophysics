# Computations in Astrophysics

A collection of computational problems in astrophysics solved numerically using MATLAB as part of a self-teaching astrophysics pathway.

# Motivation 

As a mechanical engineering undergraduate, I am self-learning astrophysics using online resources. This repository documents some of the common computational problems in astrophysics and their solutions determined using MATLAB software.

# Problems Documented

1. Free-fall behaviour of a mass

The variation of distance and velocity with respect to time was determined using RK4 and Euler's numerical methods, and the results were compared with each other for accuracy. 

2. Orbit of the Earth

The orbital path of the Earth around the Sun was plotted in two dimensions by solving Newton's gravitational formula using the RK4 numerical method.
